# FRONT-END-DEVELOPMENT-VIBE-CODING

## Hunger Point App

A modern, responsive front-end web application for Hunger Point — Shewa Adda, a food ordering and delivery service. Built with HTML, CSS, and JavaScript.

### Features

- Responsive design optimized for mobile and desktop
- Modern UI with gradient backgrounds and smooth animations
- Sticky navigation with scroll behavior
- WhatsApp integration for direct ordering
- Clean, accessible interface

### Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Google Fonts (Bebas Neue, Nunito)

### Getting Started

1. Clone the repository
2. Open `hunger-point-app.html` in your web browser
3. Enjoy the app!
